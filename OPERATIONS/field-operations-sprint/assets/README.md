# Field Assets Storage

Store captured real-world files here. **Do not commit large media to git** — use `.gitignore` for raw photos/videos.

## Folder structure

```
assets/
├── photos/
│   ├── forest-city/
│   ├── rf-princess-cove/
│   ├── danga-bay/
│   ├── bukit-indah/
│   ├── mount-austin/
│   ├── eco-botanic/
│   └── medini/
└── videos/
    ├── forest-city/
    ├── rf-princess-cove/
    └── ...
```

Log every file in `tracking/photos_log.csv` or `tracking/videos_log.csv`.
